# AR/VR App

Стек: Godot 4 (опционально Unity)

**Описание**
- Минимальное AR/VR‑приложение на Godot 4: 3D‑сцена с камерой, светом и объектом. Управление: WASD + мышь, сохранение пользовательских настроек (скорость/сенситивити) в `user://settings.json`.
- Готово к запуску локально в редакторе Godot; расширяемо до VR через OpenXR плагин; AR возможен на мобильных платформах с соответствующими плагинами.

**Установка и запуск (Godot)**
- Скачайте Godot 4.x с официального сайта.
- Откройте проект: `projects/ar-vr-app/godot/` (`project.godot`).
- Запустите сцену: `res://scenes/Main.tscn`.
- Управление:
  - Мышь: поворот камеры (захват курсора по ЛКМ, выход — `Esc`).
  - Клавиши: `W/A/S/D` — перемещение; `R` — сброс позиции/поворота; `S` — сохранить настройки.

**Unity (опционально)**
- Возможна интеграция той же сцены/логики в Unity с XR Interaction Toolkit. В текущей реализации используется Godot.

**Полезный функционал**
- Сохранение/загрузка пользовательских настроек (`user://settings.json`).
- Простая камера от первого лица для навигации по сцене.

**Локальный тест**
- Запуск выполняется из редактора Godot: сцена `Main.tscn`.
- Для VR: добавьте плагин OpenXR и соответствующую конфигурацию рендера/устройств.
