# Private Usage (RU)

Для локального использования:
- Назначение: внутренняя демонстрация AR/VR‑функций, тестирование навигации и настроек.
- Проблемы, которые решает: быстрый просмотр 3D сцены, проверка управления, базовая сохранность настроек пользователя.
- Запуск:
  - Установите Godot 4.x.
  - Откройте проект `projects/ar-vr-app/godot/project.godot`.
  - Запустите `res://scenes/Main.tscn`.
- Примечание: этот файл предназначен только для локального хранения и не должен публиковаться.
