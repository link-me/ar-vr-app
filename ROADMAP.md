# ROADMAP (RU)

- Базовый просмотр 3D сцены (камера, свет, примитивы).
- Сохранение пользовательских настроек (скорость, чувствительность).
- Импорт моделей GLTF/GLB и простая навигация.
- Режим VR через OpenXR (контроллеры, телепортация).
- Режим AR на мобильных платформах с плагинами (камера, якоря).
- Меню настроек: сенситивити, скорость, инверсия осей, пресеты.
- Экспортинг: сборки для Windows/Linux/macOS/Web.
- Локализация UI (RU/EN).
- Логирование и отладка: профайлинг рендеринга.
