# Roadmap (EN)

- Basic 3D scene viewing (camera, light, primitives).
- Persist user settings (speed, sensitivity).
- Import GLTF/GLB models and simple navigation.
- VR via OpenXR (controllers, teleportation).
- AR on mobile platforms with plugins (camera, anchors).
- Settings menu: sensitivity, speed, axis inversion, presets.
- Export builds for Windows/Linux/macOS/Web.
- UI localization (RU/EN).
- Logging and debugging: rendering profiling.
